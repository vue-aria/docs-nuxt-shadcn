## **Installation**

::code-group
```bash [npm]
npm install @aria-vue/button
```

```bash [pnpm]
pnpm add @aria-vue/button
```

```bash [yarn]
yarn add @aria-vue/button
```

```bash [bun]
bun add @aria-vue/button
```
::
